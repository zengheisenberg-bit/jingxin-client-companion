# 发布检查清单

## 上传前

- [ ] GitHub 已新建仓库
- [ ] 仓库根目录已上传 `index.html`
- [ ] 仓库根目录已上传 `share.html`
- [ ] 仓库根目录已上传 `.nojekyll`
- [ ] 仓库根目录已上传 `README.md`

## 开启 GitHub Pages

- [ ] 进入仓库 `Settings`
- [ ] 进入 `Pages`
- [ ] Source 选择 `Deploy from a branch`
- [ ] Branch 选择 `main`
- [ ] Folder 选择 `/ (root)`
- [ ] 点击保存
- [ ] 等待 1-3 分钟，直到出现 GitHub Pages 地址

## 部署后

- [ ] 手机能打开 `https://你的用户名.github.io/仓库名/index.html`
- [ ] 手机能打开 `https://你的用户名.github.io/仓库名/share.html`
- [ ] 在主系统点击「线上地址」
- [ ] 填入 `https://你的用户名.github.io/仓库名/share.html`
- [ ] 点击「共享查看」
- [ ] 复制“推荐人手机可打开链接”
- [ ] 用手机测试链接能打开
